pipeline
========

End-to-end inference orchestrator. Wires the KNN search, v6 feature
enrichment, and LightGBM re-ranker into the single ``predict``
function. Supports both unified KNN and aspect-separated KNN modes via
``PredictConfig.aspect_separated``.
