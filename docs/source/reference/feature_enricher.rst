feature_enricher
================

V6 feature enrichment for GO predictions. Computes the 25-column
feature set consumed by the post-2026 LightGBM re-ranker, including
Anc2Vec-based centroids, taxonomy voter counters, and optional PCA
projections of query embeddings.
