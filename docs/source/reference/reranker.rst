reranker
========

LightGBM re-ranker inference helpers. Exports the feature-column
schema (re-exported from ``protea_contracts``), the ``prepare_dataset``
data-preparation helper, the ``apply_reranker`` scoring function, model
serialization round-trips, and ``infer_active_feature_families``. No
FastAPI or SQLAlchemy dependency.
